📚 Array Collection

A comprehensive collection of array-based programs developed in **VB.NET**. This project demonstrates the implementation of fundamental and intermediate array concepts, helping students improve programming logic and problem-solving skills through practical examples.

---

🚀 About the Project

The **Array Collection** project is designed to provide hands-on practice with array operations in VB.NET. It includes a variety of programs that cover common array manipulation techniques, searching, sorting, and logical problem-solving exercises.

---

✨ Features

- One-Dimensional Arrays
- Two-Dimensional Arrays
- Array Traversal
- Insert & Delete Operations
- Searching Techniques
- Sorting Methods
- Reverse an Array
- Sum & Average Calculation
- Duplicate & Unique Elements
- Matrix-Based Programs
- Menu-Driven Implementation

---

🛠️ Tech Stack

- **Language:** VB.NET
- **Framework:** .NET Framework
- **IDE:** Visual Studio

  ---

🎯 Objectives

- Learn array concepts using VB.NET.
- Strengthen logical thinking and problem-solving skills.
- Practice commonly used array operations.
- Build a strong foundation for Data Structures and Algorithms.

 ---

▶️ Getting Started

1. Clone the repository

```bash
git clone https://github.com/preetichablani/Array-Collection.git
```

2. Open the solution in **Visual Studio**.

3. Build and run the project.
